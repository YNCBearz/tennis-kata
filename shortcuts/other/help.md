## [Mac]

### pbcopy : 在terminal中複製

```zsh
$ cat myfile.txt | pbcopy
```

```zsh
$ echo 'Hello' | pbcopy
```

```zsh
$ git rev-parse origin/develop | pbcopy
```

### pbpaste : 在terminal中貼上

```zsh
$ pbpaste > bear-test.md
```
